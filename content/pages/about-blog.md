# 关于这个博客

## 博客理念

这是一个专注于技术分享的个人博客，秉承"内容胜过复杂性"的理念。我相信好的内容不需要华丽的装饰，简洁清晰的设计更能让读者专注于文章本身。

## 技术栈

这个博客使用现代化的技术栈构建：

- **框架**：Next.js 14 with App Router
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **内容管理**：Markdown + gray-matter
- **部署**：Docker + Nginx

## 特性

- ✨ **现代化设计**：简洁、响应式的界面设计
- 🌙 **深色模式**：支持系统主题切换
- 📝 **Markdown支持**：原生支持Markdown写作
- 🚀 **ISR增量静态再生**：内容更新无需重新构建
- 🏷️ **标签系统**：文章分类和标签管理
- 📱 **响应式**：完美适配移动端和桌面端
- ⚡ **性能优化**：静态生成 + 图片优化

## 内容主题

这个博客主要分享以下内容：

- 开发经验分享
- 工具和工作流程优化
- 技术思考和学习笔记
- 开源项目介绍
